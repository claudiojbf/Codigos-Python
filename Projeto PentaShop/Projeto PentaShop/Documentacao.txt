TEMOS CINCO FUNÇÕES DISPONIVEIS, UMA PARA CADASTRO DE PEÇAS DE ROUPAS:

1° MASCULINAS(cadastro_masculino);
 
2° FEMININAS(cadastro_feminino);

3° INFANTIL(cadastro_infantil);

4° ACESSORIOS(cadastro_acessorios);

5° DELETAR ACESSÓRIO(deleta)


ONDE AS TRÊS PRIMEIRAS PEDIRÃO: CATEGORIA, NOME DA PEÇA, PREÇO, QUANTIDADE DE TAMANHOS, TAMANHO

O TAMANHO É DIVIDIDO EM 3 TIPOS: 

PEQUENO(P);
MEDIO(M);
GRANDE(G);

DEPENDENDO DE QUANTOS TAMANHOS SERÃO ESCOLHIDOS O FORMADO DO ARQUIVO SERÁ:

EX:

calca-jeans-10.0-P

blusa-azul-10.0-P-M


O ACESSÓRIO APENAS LHE PEDIRÁ: CATEGORIA, NOME DA PEÇA, PREÇO;

COM AS INFORMACOES PASSADAS SERÁ ADICIONADO AO ARQUIVO .TXT COMO CADA PEÇA SENDO UMA NOVA LINHA

NAO FAZEMOS ALTERAÇÃO DE VALOR APENAS EXCLUIMOS A LINHA E APOS ISSO VOCÊ TERA QUE INFORMAR NOVAMENTE OS VALORES DO PRODUTO.


A FUNÇÃO DELETAR: PEDE PARA QUE O USUÁRIO ESCOLHA O ARQUIVO QUAL DESEJA MODIFICAR E GERA
UMA LISTA DO ARQUIVO ESPECIFICO CONTENDO TODOS OS ITENS CADASTRADOS, APÓS ISSO O USUÁRIO PRECISARÁ
INFORMAR QUAL O INDICE DO ITEM QUE DESEJA DELETAR. SEGUINDO ESSES PASSOS O USUÁRIO IRÁ DELETAR A LINHA 
POR COMPLETO, PODENDO CADASTRAR O ITEM NOVAMENTE EXECUTANDO AS FUNÇÕES DE CADASTRO JÁ INFORMADAS A CIMA.
